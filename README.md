# Model Predictive Control — MATLAB and Python examples
<p align="center"> <a href="https://link.springer.com/book/9783031875953"> <img src="https://media.springernature.com/w600/springer-static/cover-hires/book/978-3-031-87595-3" alt="Model Predictive Control — Third Edition (Springer cover)" width="280"> </a> </p>
This repository contains MATLAB example codes accompanying the book “Model Predictive Control (Third Edition)” (Springer).
Book page: https://link.springer.com/book/9783031875953

## Contents
Matlab and python files corresponding to some of the examples of the book

## Requirements
MATLAB R2020b or newer (older versions may work).
Optimization Toolbox (e.g., for quadprog) if required by your scripts.
MPC Toolbox (optional), only if some examples depend on it.

## How to run
Clone or download this repo.
Open MATLAB and add the folder to the path.
Run!
